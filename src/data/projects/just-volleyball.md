# 專案介紹

## 起源

本專案起源於我們班要收手機（挺慘的），於是大家下課無聊到彷彿回到國中時候，就有人用前面的電腦玩 **皮卡丘排球：https://gorisanson.github.io/pikachu-volleyball/zh/**
。

## 開發過程

沒錯，啊我看著看著就想說，不如我也來做一個打排球小遊戲好了，於是我先做了一個單人本地單挑版本，接下來因為太多人想玩(_應該是這樣吧_)，所以我就弄了一個計時 + 計分雙決勝負版本，然後又加了一個雙人版本，可是問題就在於說如果我都在同一台電腦我需要 3 \* 4 個按鈕，到最後一堆人卡在一台筆電前面根本不知道在玩啥，太卡了，也因此再來我加入一個區網模式，就是可以一起連學校網路連線玩，對但是那個版本無法丟上來，而且應該也刪掉了。總之後來可能就再優化一些有的沒的，就是最後在 github 上面的版本了，啊底下的連結是去到我的專案網址，而非遊玩網站，要玩的話要點右上角有一個連結進去。
